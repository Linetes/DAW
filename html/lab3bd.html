MariaDB

Requerimientos Generales de Hardware

MariaDB doesn't have minimal hardware requirements per se, or if there was any, it would correspond to pre-2000 machines which don't exist anymore. So it mostly depends on what you intend to do with it. Tarball is approximately 400MB, by my last count. Packages are much less than that.

Ventajas y desventajas de su uso

Ventajas Desventajas
Supports significantly more engines (SphinxSE, Aria, FederatedX, TokuDB, Spider, ScaleDB and etc.) A subsequent migration of MariaDB back to MySQL has not been possible since release 5.5.36
Developing more continuously compared to MySQL Required upgrade to a newer version due to dependencies
Offers a cluster database for commercial use, which also enables multi-master replication. Anyone can use it freely and reliance on a MySQL Enterprise system is not required. Errors occur more often

Conclusión

En conclusión, el uso de Manejadores de Base de Datos Relacionales es algo caro cuando se usa de modo profesional. Creo que conviene usar SQL Server para uso de developer ya que es gratis. Oracle es el mejor pero es muy caro y creo que sólo es viable cuando ya tienes una empresa y buscas lo mejor.
