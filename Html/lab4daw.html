¿Qué diferencias y semejanzas hay entre Java y JavaScript?

Java es un lenguaje de programación orientado a objetos, mientras JavaScript es un lenguaje de "scripting" orientado objetos. Java crea aplicaciones en una máquina virtual, mientras JavaScript funciona en el browser. Java es de Oracle, mientras que JavaScript de Netscape. En otras palabras, lo único que tienen en común es el nombre "Java". Todo lo demás es diferente.

¿Qué métodos tiene el objeto Date?
  • getDate() - Regresa el día del mes
  • getDay() - Regresa el día de la semana
  • getFullYear() - Regresa el año
  • getHours() - Regresa la hora
  • now() - Regresa el número de milisegundos desde el 01/01/1970
  • setDate() - Pone el día del mes que gustes
¿Qué métodos tienen los arreglos?
  • concat() - Concatena 2 arreglos
  • every() - Checa si cada elemento del arreglo pasa una prueba
  • fill() - Llena los elementos en un arreglo con un valor estático
  • filter() - Filtra los elementos de un arreglo con una prueba
  • find() - Regresa el primer elemento de un arreglo que pasa una prueba
  • indexOf() - Busca en el arreglo un elemento y te regresa la posición
¿Cómo se declara una variable con alcance local dentro de una función?

Se tiene que declarar dentro de la función y se declara como:

var NAME = "";

¿Qué implicaciones tiene utilizar variables globales dentro de funciones?

Que se pueden usar en cualquier función y que si el valor es modificado, ese se guarda globalmente.

¿Qué método de String se puede utilizar para buscar patrones con expresiones regulares?¿Para qué podrías utilizar esto en una aplicación web?

exec() - Hace la prueba si hace match el string y regresa el primer match o test() - Hace la prueba si hace match el string y regresa verdadero o falso. Se podría usar para la barra de Search dentro de una aplicación web.
