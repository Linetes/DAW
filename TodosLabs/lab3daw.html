Como ingeniero de software ¿cuál es tu recomendación sobre el uso de !important en un CSS?

Es muy recomendable su uso si tienes varios CSS o si usas un framework porque le pone prioridad a lo que quieres. Si quieres combinar algo como paragraph con algo que tenga id para darle formato, lo que tenga important va a tener prioridad.

Si se pone una imagen de fondo en una página HTML, ¿por qué debe escogerse con cuidado?

El fondo de pantalla puede ser un problema si no se cuida porque no todas las pantallas tienen la misma definición. También en términos de diseño, un fondo muy brillante o deslumbrante puede afectar a la lectura de la página. También es difícil ajustar el tamaño de las imágenes, es mejor editarla antes.

Como ingeniero de software, ¿cuál es tu recomendación al elegir las unidades de un propiedad de estilo entre %, px y pt?

Creo que lo mejor es usar % ya que px y pt ya tienen un tamaño preterminado y las resoluciones de pantallas varían, así que para una compu 20 px puede ser muy grande y para otra muy pequeño, mientras % se adapta a la resolución.

¿Por qué el uso de una versión minimizada del CSS mejora el rendimiento del sitio?

Mejora el rendimiento porque reduce el tamaño del archivo borrando los white-spaces y comentarios. Esto hace que la página se carge más rápido.
