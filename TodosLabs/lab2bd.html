Cuestionario 1: BD vs DBMS

BD vs DBMS

Qué es entonces una base de datos - DB?

Una Base de Datos se puede pensar como una colección de archivos que están conectados.

Es muy importante tener bien diseñada una Base de Datos para que haya la menor cantidad de repeticiones.

¿En qué casos es conveniente usar bases de datos?

Es conveniente usar una Base de Datos cuando hay manejo de información. También es importante saber el tamaño y posible extensión de la información, si es para algo chico se pueden usar archivos. Si se necesita usar por varios usuarios o aplicaciones, si la extensión de el número de registros es grande, lo mejor es usar una base de datos.

¿Qué es un sistema de gestión de base de datos - DBMS?

Un sistema de gestión de base de datos es un conjunto de archivos conectados y uno o varios programas para acceder a dichos archivos.

El sistema de gestión de base de datos proporciona una forma organizada de acceder a la base de datos.

Enlista y explica las funciones/responsabilidades que tienen los DBMS.

  • Interacción con el gestor de archivos: Permite la edición y acceso a las bases de datos.
  • Implantación de la integridad: Satisfacer ciertos tipos de restricciones de consistencia.
  • Implantación de la seguridad: Dar acceso a ciertas personas con roles.
  • Copia de seguridad y recuperación: Pueden haber fallos y perderse info. Por eso el gestor detecta fallos y respalda la base de datos.
  • Control de concurrencia: Conservar consistencia de datos aunque hayan varios usuarios editándola constantemente.

Ejemplifica y justifica en que proyecto de los que has realizado hubiera sido conveniente utilizar una base de datos.
